## 查询下压配置

本产品内嵌了一个智能查询下压引擎（Smart Pushdown Engine），支持各种 SQL on Hadoop 引擎，如 Hive、SparkSQL、Impala，同时也支持传统数据库/数据仓库，如Greenplum、Netezza、SQL Server。

Kyligence Enterprise 在必要时可以将 Cube 不适合的查询语句，路由到查询下压引擎中。
