## Arithmetic Operators

| Operator | Description                               | Syntax | Example              |
| -------- | ---------------------------------------- | ------ | -------------------- |
| +        | Plus                                     | A+B    | Cost+Profit          |
| -        | Minus.                                   | A-B    | Revenue- Cost        |
| *        | Multiply. Returns *numeric1* multiplied by *numeric2* | A*B    | Unit_Price* Quantity |
| /        | Divide                                   | A/B    | Total_Sale/Quantity  |
